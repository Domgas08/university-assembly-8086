## Character Frequency Counter (8086 Assembly)

This program reads a text file and counts how many times each character appears.
The results are written to an output file as the character's ASCII code (in hex system) and the number of occurrences.

Constraints:
- Each character can appear at most 255 times

### Example:
Input file (`duom.txt`):
abca

Output file (`rez.txt`):
61 2
62 1
63 1
